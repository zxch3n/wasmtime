void wasmtime_error_message ( const wasmtime_error_t *  error,
wasm_name_t *  message

Returns the string description of this error.

This will "render" the error to a string and then return the string representation of the error to the caller. The message argument should be uninitialized before this function is called and the caller is responsible for deallocating it with wasm_byte_vec_delete afterwards.
