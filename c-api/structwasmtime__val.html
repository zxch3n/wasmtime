Detailed Description

Container for different kinds of wasm values.

Note that this structure may contain an owned value, namely wasmtime_externref_t, depending on the context in which this is used. APIs which consume a wasmtime_val_t do not take ownership, but APIs that return wasmtime_val_t require that wasmtime_val_delete is called to deallocate the value.
